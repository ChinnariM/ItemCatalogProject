Install Vagrant and VirtualBox
Clone the fullstack-nanodegree-vm
Launch the Vagrant VM (vagrant up)

1# Create a Data base structure
Create a database_setup.py file
then run it as a python database_setup.py

it should create a catalog empty database.
2# Insert some data in to the created catagory database.
Create a lotsofitems.py file
then run it as a python lotsfitems.py
it should insert data to the Catagory and Items tables

3# now create the webservers to call the data in the UI
4# now run the application.py once it starts user can interact with the application with localhost:5000 URL.
